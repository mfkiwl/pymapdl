ansys.mapdl.core.post.PostProcessing.plot_nodal_elastic_strain_intensity

PostProcessing.plot_nodal_elastic_strain_intensity(show_node_numbering=False, **kwargs)

Plot the elastic nodal strain intensity of the current result.

Returns

cpos Camera position from plotter. Can be reused as an input parameter to use the same camera position for future plots.

Return type

list

Examples

Plot the elastic strain intensity for the second result

>>> mapdl.post1()
>>> mapdl.set(1, 2)
>>> mapdl.post_processing.plot_nodal_elastic_strain_intensity()

Plot off_screen and save a screenshot

>>> mapdl.post_processing.plot_nodal_elastic_strain_intensity(off_screen=True,
                                                            savefig='seqv_00.png')

Subselect a single result type and plot those strain results >>> mapdl.esel(S, TYPE, vmin=1) >>> mapdl.post_processing.plot_nodal_elastic_strain_intensity()
